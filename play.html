💡 How to get Cloudnestra URLs:

Method 1 - From Video Player:
  1. Open browser Developer Tools (F12)
  2. Go to "Network" tab → Filter by "Media"
  3. Play the video and look for .m3u8 or .mp4 requests
  4. Right-click → Copy → Copy URL
Method 2 - From Page Source:
  1. Right-click on video page → View Page Source
  2. Search for "cloudnestra.com/rcp/"
  3. Copy the full URL
Method 3 - From iframe src:
  1. Inspect the video iframe element
  2. Look for src attribute containing cloudnestra.com
  3. Copy that URL
